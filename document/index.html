Dialog

创建一个Dialog对象

new Dialog(options: object)
Parameters
options (object) 用户配置
Example
var dialog = new Dialog({
		title: 'hi',
		content: 'hello eui-dialog'
	});

mixin

对象属性混入

mixin
Parameters
destination (object) 需要混入到此对象中
source (object) 待混入的对象
Returns
object: 混入的最终结果，就是改变后的destination
Example
$dialog.mixin({ a:1, b:2 }, { c:3, a:0 })
//-> { a:0, b:2, c:3 }
